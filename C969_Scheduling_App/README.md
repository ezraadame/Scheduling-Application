# Scheduling Application

A Windows Forms desktop application built with C# and .NET Framework for managing customers and appointments with user authentication.

## Prerequisites

- Windows OS
- .NET Framework 4.8 or higher
- MySQL Community Server 9.5+

## Technologies Used
- C#
- Windows Forms
- MySQL

## Features
- User Authentication: Secure login system for users.
- Customer Management: Add, update, delete, and view customer information.
- Appointment Management: Schedule, update, delete, and view appointments.
- Data Validation: Ensures data integrity and correctness.
- Reporting: Generate reports on appointments and customers.
- Time Zone Handling: Manages appointments across different time zones.
- Localization: Supports multiple languages for a broader user base.
- Error Handling: Robust error handling and logging mechanisms.
- User-Friendly Interface: Intuitive and easy-to-navigate UI.
- Database Integration: Seamless integration with MySQL for data storage and retrieval.
- Notifications: Alerts for upcoming appointments.
- Search Functionality: Easily search for customers and appointments.

## Installation
1. Users will need to have MySQL Community Server 9.5 or higher installed on their machine.
2. Clone the repository to your local machine.
3. Open the solution in Visual Studio.
4. Restore NuGet packages if necessary.
5. Update the database connection string in the configuration file with your MySQL server details.
	- Once you have MySQL installed, you will need to create a database and import the provided SQL script located in the `Database` folder of the project.
6. Build and run the application.
